# tcping
使用tcp的ping，支持很多功能
## 预备
[下载](https://github.com/intmian/tcping/releases/download/v1.0/tcping.exe)
放入系统环境里面的任意path文件夹
使用
## 使用方法
tcping.exe (-a addr [-p port] [-n testNum] [-pr (tcp||udp)]) || (-h)
## 参数详情
>-a    地址
-h    帮助
-n    测试次数 (default 5)
-p    端口 (default "80")
-pr   协议(tcp/udp) (default "tcp")
